Back to Home

Description: Fitting involves assembling various machine parts together by using different tools and techniques to achieve a functional unit.

Apparatus Used:

  • Fitting Bench
  • Vices
  • Files
  • Hacksaws
  • Drill Machine
  • Calipers
  • Surface Plate
  • Angle Plate

Process:

  1. Mark the workpiece using a marking gauge.
  2. Cut the material to size using a hacksaw.
  3. File the workpiece to the desired shape and size.
  4. Drill holes as required.
  5. Check the fitting of the parts using calipers and surface plate.
